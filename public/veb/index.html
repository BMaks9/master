Типы задач:
1. Назначить цвет

Ответ:

Можно так:
DIV#section1 DIV.class1 DIV {
    color: green;
}

А можно вот так:
DIV#section1 > DIV.class1 > DIV {
    color: green;
}

И так тоже:
#section1 > .class1 > DIV {
    color: green;
}

А еще так:
#section1 .class1 DIV {
    color: green;
}

И даже так:
*#section1 > *.class1 > DIV {
    color: green;
}
*#section1 *.class1 DIV {
    color: green;
}
  1. Тоже выбрать цвет, но чуть-чуть по другому:

Ответ:

DIV#section1 > DIV.class1 > DIV.class2 > DIV {
    color: green;
}
#section1 .class1 .class2 > DIV {
    color: green;
}
*#section1 > *.class1 > *.class2 > DIV {
    color: green;
}
  1. Два цвета

Ответ:

Вот так пойдет:
#section1 > .class1 > H1.main {
    color: green;
}
#section1 > .class1 > P.main {
    color: blue;
}

А можно и так:
DIV#section1 DIV.class1 H1.main{
    color: green;
}
DIV#section1 DIV.class1 P.main{
    color: blue;
}
  1. Хитрый

Ответ:

#section1 > .class1 > .class2 > H1.main {
    color: blue;
}

#section1 > .class1 > .class2 > P.main {
    color: red;
}

Можно указать типы элементов:
SPAN#section1 > SPAN.class1 > SPAN.class2 > H1.main {
    color: blue;
}

SPAN#section1 > SPAN.class1 > SPAN.class2 > P.main {
    color: red;
}
  1. Угадать цвет:

Ответ:
При media = "screen", текст будет отображаться на экране, соответственно подключится style.css.
Тогда будем иметь следующие стили для P:

P.P_CLASS {color: blue;}
P {color: brown;}
P {color:red;}
P {color: green;}

Соответственно P.P_CLASS {color: blue;} - более спецефичный, так как имеет один класс и один элемент, а все остальные только один элемент.

В итоге будет СИНИЙ

При media = "print", стиль предназначен только для печати, style.css не подключится, будут стили:

P {color:red;}
P {color: green;}

Применится тот, который идет последним, будет ЗЕЛЕНЫЙ
  1. На закрепление:

Ответ:

Когда media="screen", применится style2.css
Имеем:
P.P_CLASS {color: blue;}
P {color: green;}
P.P_CLASS {color: red;}

Одинаковая спецефичность у P.P_CLASS {color: blue;} и P.P_CLASS {color: red;}, но P.P_CLASS {color: red;} идет позже, он и применится, будт КРАСНЫЙ

Когда media="print", style2.css не применится 
Имеем:
P.P_CLASS {color: blue;}
P {color: green;}

Будет СИНИЙ, так как P.P_CLASS {color: blue;} более спецефичный
