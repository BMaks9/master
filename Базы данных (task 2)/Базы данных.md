Типы задач:
1. Простоя выборка из 1 таблицы
![[Pasted image 20260705194918.png]]

Решение:
```
SELECT model, ram, screen FROM Laptop 
WHERE price > 1000 or price = 500 
ORDER BY ram;
```
$$ \tau_{ram}(\pi_{model, ram, screen}(\sigma_{price > 1000 \lor price = 500}(Laptop))) $$

2. Выборка по двум таблицам
![[Pasted image 20260705195150.png|682]]

Решение:
```
SELECT DISTINCT Product.maker FROM Product
JOIN PC ON Product.model = PC.model
WHERE PC.speed >= 450
ORDER BY Product.maker;
```
$$
\tau_{maker}(\pi_{maker}(\sigma_{speed>=450}(Product\bowtie_{Product.model=PC.model}PC)))
$$
3. Найти максимальную цену
![[Pasted image 20260705195336.png]]

Решение:
```
SELECT model, price FROM Printer
WHERE price = (SELECT MAX(price) FROM Printer)
ORDER BY price;
```
$$
\tau_{price}(\pi_{model, price}(\sigma_{price = max\_price}(Printer \times \gamma_{max(price) \rightarrow max\_price}(Printer))))
$$
4. Найти среднее
![[Pasted image 20260705195453.png]]

Решение:
```
SELECT AVG(PC.speed) AS avg_speed FROM PC
JOIN Product ON PC.model = Product.model
WHERE Product.maker = 'A';
```
$$
\gamma_{avg(speed) \rightarrow avg\_speed}(\sigma_{maker = 'A'}(PC \bowtie_{PC.model = Product.model} Product))
$$
5. С подзапросом + JOIN
![[Pasted image 20260705195624.png]]

Решение:
```
SELECT p.type, l.model, l.speed FROM Laptop AS l
JOIN Product AS p ON l.model = p.model
WHERE l.speed < (SELECT MIN(speed) FROM PC);
```

Дополнительно ебанем на реляционном языке
$$
\pi_{type, model, speed}(\sigma_{speed < min\_speed}((Laptop \bowtie_{Laptop.model = Product.model} Product) \times \gamma_{min(speed) \rightarrow min\_speed}(PC)))
$$
6. JOIN + агрегатор
![[Pasted image 20260705200006.png]]

Решение:
```
SELECT p.maker, AVG(l.screen) as avg_screen FROM Product AS p
JOIN Laptop AS l ON p.model = l.model
GROUP BY p.maker
ORDER BY avg_screen;
```

$$
\tau_{avg\_screen}(\gamma_{maker;avg(screen)\rightarrow avg\_screen}(Product \bowtie_{Product.model=Laptop.model}Laptop))
$$

Справка:
Ответ на вопрос состоит из двух частей, 1 - составить запрос на SQL, 2 - написать запрос на языке реляционной алгебры.

Основные операции реляционной алгебры:
$$
\begin{array}{l}
\sigma - \text{ выборка, аналог SELECT по горизонтали} \\
\pi - \text{проекция, аналог SELECT по вертикали} \\
\tau - \text{сортировка ORDER} \\
\bowtie - JOIN \\
\gamma - агрегирующая функция \\
\rightarrow - переименование
\end{array}
$$
