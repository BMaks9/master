Типы задач:
1. Назначить цвет
![[Pasted image 20260713225307.png]]

Ответ:
```
Можно так:
DIV#section1 DIV.class1 DIV {
	color: green;
}

А можно вот так:
DIV#section1 > DIV.class1 > DIV {
	color: green;
}

И так тоже:
#section1 > .class1 > DIV {
	color: green;
}

А еще так:
#section1 .class1 DIV {
	color: green;
}

И даже так:
*#section1 > *.class1 > DIV {
	color: green;
}
*#section1 *.class1 DIV {
	color: green;
}
```

5. Угадать цвет:
![[Pasted image 20260713233155.png]]

Ответ:
При media = "screen", текст будет отображаться на экране, соответственно подключится style.css.
Тогда будем иметь следующие стили для P:
```
P.P_CLASS {color: blue;}
P {color: brown;}
P {color:red;}
P {color: green;}

Соответственно P.P_CLASS {color: blue;} - более спецефичный, так как имеет один класс и один элемент, а все остальные только один элемент.

В итоге будет СИНИЙ
```

При media = "print", стиль предназначен только для печати, style.css не подключится, будут стили:
```
P {color:red;}
P {color: green;}

Применится тот, который идет последним, будет ЗЕЛЕНЫЙ
```

6. На закрепление:
![[Pasted image 20260713235009.png]]

Ответ:
```
Когда media="screen", применится style2.css
Имеем:
P.P_CLASS {color: blue;}
P {color: green;}
P.P_CLASS {color: red;}

Одинаковая спецефичность у P.P_CLASS {color: blue;} и P.P_CLASS {color: red;}, но P.P_CLASS {color: red;} идет позже, он и применится, будт КРАСНЫЙ

Когда media="print", style2.css не применится 
Имеем:
P.P_CLASS {color: blue;}
P {color: green;}

Будет СИНИЙ, так как P.P_CLASS {color: blue;} более спецефичный
```
